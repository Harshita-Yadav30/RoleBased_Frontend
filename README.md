# Role-Based Dashboard App

A full-stack web application with role-based access control built using **Next.js (frontend)** and **Node.js + Express + MongoDB (backend)**. Users can create, edit, and delete items, while admins can manage users and assign roles.

---

## Features

### User Features
- Sign up / Login
- Create, edit, and delete items
- Search and paginate items

### Admin Features
- Access admin panel
- View all users
- Filter users by role
- Update user roles (Admin/User)

---

## Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS
- **Backend:** Node.js, Express, MongoDB, Mongoose
- **Authentication:** JWT (JSON Web Tokens)
- **Deployment:** Vercel (frontend), Render/Railway (backend)
